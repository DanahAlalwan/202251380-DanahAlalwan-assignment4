
# AI Usage Report

## Tools Used & Use Cases

### ChatGPT
I used ChatGPT to:
- brainstorm improvements for the final portfolio website
- refine the structure of the HTML, CSS, and JavaScript files
- improve UI/UX ideas for a more professional layout
- review code quality and suggest better organization
- help write project documentation

### GitHub Copilot (optional if you used it)
I used GitHub Copilot to:
- speed up writing repetitive code
- suggest code completions while developing
- help generate small code blocks during implementation

## Benefits & Challenges

### Benefits
- Helped me improve the structure and readability of the code
- Gave ideas for making the website more polished and interactive
- Saved time while writing documentation and debugging
- Helped me understand better ways to validate forms and organize features

### Challenges
- Some generated suggestions needed editing to match my project exactly
- I had to review all AI suggestions carefully before using them
- Some ideas were too advanced or unnecessary, so I simplified them

## Learning Outcomes
By using AI tools in this assignment, I learned:
- how to improve code structure and maintainability
- how to build a more professional user interface
- how to review AI-generated content critically
- how to combine technical development with proper documentation

## Responsible Use & Modifications
I did not copy AI-generated output directly without review. I checked, modified, and adapted all suggestions to fit my own project requirements. I made sure I understood the logic of the code before including it. I also revised the wording and structure of the documentation to reflect my actual work and learning process.
