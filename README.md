# Assignment 4 – Personal Web Application

## Project Description
This project is the final version of my personal portfolio web application for Assignment 4. It combines all the concepts learned throughout the course into one polished and professional website.

The website presents my profile, skills, featured projects, live GitHub repositories, personalized greeting, and contact form. It is responsive, interactive, and designed to provide a clean user experience across different devices.

## Features
- Responsive portfolio website
- Dark/light mode with saved preference using `localStorage`
- Mobile navigation menu
- Typing effect in the hero section
- Scroll progress bar
- Project filtering by category
- Project sorting by title and date
- Project search functionality
- Project details modal
- Live GitHub repositories fetched using the GitHub API
- Visitor name storage using `localStorage`
- Contact form validation
- Professional documentation

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- GitHub REST API
- Git & GitHub
